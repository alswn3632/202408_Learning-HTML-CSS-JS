날짜 관련 함수

  • new Date() => 오늘 날짜 반환 new Date(String => String 날짜 형식으로 변환)
  • getFullYear(); 연도만 리턴
  • getMonth(); => 0월부터 11월까지 **getMonth()+1; '+1'을 반드시 해줘야함.
  • getDate(); => 1일부터 31일까지
  • getDay(); => 요일을 리턴 0(일)부터 6(토)까지.
  • getHours(), getMinutes(), getSeconds() : 시분초 추출
  • toLocaleDateString() : 2024. 8. 7. toLocaleTimeString() : 오전 10:01:35 toLocaleString() : 2024. 8. 7. 오전 10:01:40
