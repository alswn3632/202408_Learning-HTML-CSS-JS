string : 가장 많이 다루는 데이터 타입

  • java에서 사용하는 String method와 거의 유사함.
  • 문자열 데이터는 index(0부터 시작)를 가지고 있다.
  • 문자열 값 내에서 (br) 개행 하는 것을 추천하지 않는다. (데이터를 가져오기 힘들어짐)
  • new String() : 객체 타입으로 선언이 가능하다. 하지만 추천하지 않음.
  • length : 문자열의 총 길이
  • 추출 관련 메서드 : substring(), substr(), slice()
  • 문자 대체 관련 메서드 : replace(old,new) => 처음 찾은 값만 사용 가능
  • trim() : 양쪽 공백 제거
  • padStart(길이, 채울문자) / padEnd(길이,채울문자) 길이 만큼 문자를 채워주는 함수 ex. 010101-4***** / **시 데이터는 실제로 들어가더라도 화면에 공개되지않도록 하는 경우가 있기 때문에 유의깊게 봐야 한다.
  • charAt(index) : 한글자 추출 / indexOf(문자) : 해당 문자의 위치 리턴
  • startWith(문자) / endWith(문자) 해당 문자 앞, 뒤에 문자가 존재하는지 확인하는 메서드 return true/false
