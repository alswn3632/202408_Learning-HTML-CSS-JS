element 속성은 크게 두 가지로 구분 -태그의 속성 -CSS(style) 속성

  • 태그의 속성은 태그가 가지고 있는 속성이기 때문에 속성명을 그대로 사용 li.id input.value a.href img.src
  • class는 여러개의 값을 갖는 속성 / 이런 특징을 가지는 속성들은 별도의 속성명이 있음
  • object.classList : 모든 클래스를 나열
  • CSS 관련 속성은 style. 을 통해 프로퍼티에 접근 가능
  • CSS 속성 중 - font-size => -를 지우고 첫글자를 대문자로 변경해서 사용 fontSize
