location: location 객체는 브라우저의 주소표시줄 입력에 관련된 객체

  • window.location / window 생략가능
  • **location.href = 'url' : 해당 url로 이동 (history에 저장)
  • location.replace('url') : 해당 url로 이동 / history 객체에 저장하지 않고 이동
  • location.assign('url') : 해당 url로 이동 / history 객체에 저장
  • **location.reload() : 현재 페이지에서 새로 고침
