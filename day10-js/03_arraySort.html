배열의 정렬은 문자 기반으로 정렬됨

  • array.sort() : 문자를 기준으로 오름차순
  • array.reverse() : 문자를 기준으로 내림차순
  • 정렬할 배열의 값이 숫자일 때 콜백 함수를 이용하여 내부 연산을 통해 정렬 (콜백 함수 : 메서드 안에서 function을 부르는 것)
  • [1,6,3,5,8,2] array.sort(function(a,b)){ return a-b; //오름차순 return b-a; //내림차순 (값이 -면 앞으로, +면 뒤로 보냄 => 을 이용하여 오름차순,내림차순 정렬을 할 수 있음.) }
