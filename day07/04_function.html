함수(function) : 특정 기능을 수행하기 위해 작성한 코드

  • 함수는 호출되어야만 실행 (자바의 method와 같은 의미)
  • 자바는 class => method 생성
  • 자바스크립트는 어디서는 function 생성 가능
  • function 이라는 키워드를 사용하여 정의
  • function 함수명(파라미터){ 로직 작성; return 값; }
  • 함수명 : 기능을 함축하는 의미로 작성
  • 파라미터(매개변수) : 선택적으로 작성 (없어도 ()는 있어야함)
  • 자바스크립트에서는 파라미터의 자유도가 매우 높다. 다양한 형태로 사용이 가능하다.
  • return 함수 종료 역할과 동시에 특정 값을 호출한 위치로 반환
